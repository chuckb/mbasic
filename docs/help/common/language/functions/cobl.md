# COBL

## Syntax

```basic
COBL(X)
```

**Versions:** Extended, Disk

## Description

Converts X to a double precision number.

## Example

```basic
10 A = 454.67
             20 PRINT A:COBL(A)
             RUN
              454.67 454.6700134277344
             Ok
```
